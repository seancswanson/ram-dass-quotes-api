This app provides several API endpoints to GET a payload of Ram Dass resources of various categories.

Here is the list of endpoints:

Quotes

  • api/quotes/all
  • api/quotes/love
  • api/quotes/inspiration
  • api/quotes/relationship
  • api/quotes/death
  • api/quotes/other
  • api/quotes/random

Foundations

  • api/foundations/all

This API is for educational purposes only. The dev of this site is not affiliated with Ram Dass or any of the related foundations mentioned in the web app that consumes this API: Reflections With Ram Dass.
